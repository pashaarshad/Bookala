Arshad_install_app - Installer helper for Bookala

Files included:
- BUILD_APK.bat    -> Batch script to build release APK or App Bundle (adjust flutter path if needed)
- EASY_INSTALL_STEPS.txt -> Quick install steps to get APK onto an Android phone
- HOW_TO_BUILD_APK.md -> Full instructions for installing prerequisites and building signed APK
- README.txt       -> This file

Usage:
1) Ensure Flutter and Android SDK are installed (see HOW_TO_BUILD_APK.md)
2) Run BUILD_APK.bat from this directory to build the APK
3) Use EASY_INSTALL_STEPS.txt to copy/install APK to phone

If you want me to try building the APK here, install Android SDK on this machine (or provide its path) and I will run the build and place the APK inside this folder.
